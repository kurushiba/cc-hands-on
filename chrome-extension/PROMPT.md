# Chrome拡張ハンズオン 使用プロンプト

## プロンプト① — エラー読み取り・修正・確認（最初に使う）

```
@browser 現在のページのコンソールエラーや警告をすべて確認してください。
エラーの内容を分析し、それぞれの原因とコードのどこに問題があるかを説明してください。
その上で、すべてのエラーを修正してください。
修正が完了したら @browser でコンソールを再確認し、エラーや警告が解消されているかチェックしてください。
エラーが残っていれば、すべて解消されるまでこのサイクルを繰り返してください。
```

---

## プロンプト② — 途中確認（修正後に単独で使う）

```
@browser コンソールにエラーや警告はまだ残っていますか？
残っている場合は内容を教えてください。
```
